## II.   Application Shortcuts:

### Tools:
  - `A` Artboard Tool
  - `V` Select Tool
  - `R` Rectangle Tool
  - `T` Text Tool

### Working with elements:
  - `cmd+L` lock assets to artboard, repeat to unlock
  - `shift+cmd+[` send to back
  - `cmd+R` repeat grid
  - `cmd+click` direct-select any element
